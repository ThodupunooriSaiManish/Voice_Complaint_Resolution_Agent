# Voice Complaint Resolution Agent

## Overview

The **Voice Complaint Resolution Agent** is a multi-agent AI system that automates the complaint registration and resolution workflow using Speech-to-Text (STT), Large Language Models (LLMs), and intelligent task orchestration.

The system accepts voice-based complaints from users, converts speech into text, analyzes the complaint using AI agents, categorizes the issue, determines its priority, and automatically generates a structured complaint ticket for further processing.

This project demonstrates the integration of modern AI technologies to build an intelligent complaint management system that reduces manual effort and improves response efficiency.

---

## Features

- Voice-based complaint registration
- Speech-to-Text conversion
- AI-powered complaint analysis
- Multi-agent workflow orchestration
- Automatic complaint categorization
- Intelligent priority assignment
- Automated ticket generation
- Modular and scalable architecture

## System Workflow

1. User submits a complaint through voice input.
2. Speech-to-Text converts audio into text.
3. AI agents analyze the complaint.
4. Complaint category and priority are determined.
5. A structured complaint ticket is generated.
6. The processed complaint is returned for further resolution.

## Installation

Clone the repository

```bash
git clone https://github.com/ThodupunooriSaiManish/Voice_Complaint_Resolution_Agent.git
```

Move into the project directory

```bash
cd Voice_Complaint_Resolution_Agent
```

Install the dependencies

```bash
pip install -r requirements.txt
```

---

## Running the Project

Execute

```bash
python run.py
```

The application will start and initialize the Voice Complaint Resolution Agent.
